# Academic Vocabulary Upgrades

Replace informal vocabulary with formal academic alternatives.

## Core Replacements

| Avoid | Use Instead |
|-------|-------------|
| shows | demonstrates, indicates, reveals |
| very good | effective, substantial, significant |
| get | obtain, achieve, acquire |
| thing | factor, component, element, aspect |
| big | significant, substantial, considerable |
| small | negligible, minor, minimal |
| a lot of | numerous, substantial, considerable |
| look into | investigate, examine, analyze |
| find out | determine, ascertain, discover |
| make sure | ensure, verify, confirm |
| set up | establish, configure, implement |
| carry out | conduct, perform, execute |
| come up with | develop, devise, formulate |
| deal with | address, handle, manage |
| point out | indicate, highlight, emphasize |
| put forward | propose, suggest, present |
| take into account | consider, account for |
| in order to | to (simpler is better) |
| due to the fact that | because, since |
| a number of | several, numerous |
| in the event that | if |
| at this point in time | currently, now |
| prior to | before |
| subsequent to | after |
| utilize | use (simpler is often better) |
| methodology | method (unless specifically discussing methodology) |

## Verb Strength Upgrades

| Weak | Strong |
|------|--------|
| is | represents, constitutes |
| has | possesses, exhibits, demonstrates |
| does | performs, executes, accomplishes |
| makes | produces, generates, creates |
| uses | employs, utilizes, applies |
| gives | provides, offers, yields |
| takes | requires, necessitates |
| goes | proceeds, progresses |
| comes | emerges, arises, derives |
| seems | appears, suggests |

## Hedging Language

Use appropriately for academic caution:

| Too Strong | Hedged Alternative |
|------------|-------------------|
| proves | suggests, indicates |
| definitely | likely, probably |
| always | typically, generally |
| never | rarely, seldom |
| must | may, might, could |

## Transition Upgrades

| Informal | Formal |
|----------|--------|
| Also | Furthermore, Moreover, Additionally |
| But | However, Nevertheless, Nonetheless |
| So | Therefore, Consequently, Thus |
| And | Additionally, Furthermore |
| First | Initially, Primarily |
| Next | Subsequently, Following this |
| Finally | Ultimately, In conclusion |
| Because | Since, Given that, As |

## Phrases to Avoid

- "I think" → "It is argued that" / "This study suggests"
- "We found" → "The results indicate" / "The analysis reveals"
- "You can see" → "It can be observed that"
- "This shows" → "This demonstrates" / "This indicates"
- "As you know" → Remove entirely
- "Obviously" → "Evidently" or remove
- "Basically" → Remove entirely

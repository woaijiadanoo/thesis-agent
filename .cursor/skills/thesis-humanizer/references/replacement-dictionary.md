## Connector Variations

Replace formulaic transitions with varied alternatives.

### Additive Connectors

| Overused | Alternatives |
|----------|-------------|
| Furthermore | Also, In addition, [remove] |
| Additionally | Also, Moreover, [remove] |
| Moreover | Also, Further, In addition |
| In addition | Also, [remove], Further |

### Contrastive Connectors

| Overused | Alternatives |
|----------|-------------|
| However | Yet, But, Still, Nevertheless |
| On the other hand | Conversely, Alternatively |
| Nevertheless | Yet, Still, Even so |
| Nonetheless | Still, Yet, Even so |

### Causal Connectors

| Overused | Alternatives |
|----------|-------------|
| Therefore | Thus, Consequently, Hence |
| As a result | Consequently, Thus |
| Consequently | Thus, Therefore, Hence |
| Hence | Thus, Therefore |

### Sequential Connectors

| Overused | Alternatives |
|----------|-------------|
| Firstly/Secondly/Thirdly | First/Second/Third, [remove] |
| Subsequently | Then, Later, Next |
| Finally | Lastly, In conclusion |

### Connector Removal Strategy

Often, the best replacement is removal. Ideas should connect through meaning, not forced transitions.

```
Before: "The protocol ensures privacy. Furthermore, it provides security. 
        Additionally, scalability is maintained."

After:  "The protocol ensures privacy and security while maintaining 
        scalability."
```

## Chapter Organisation Verbs

Vary verbs when introducing chapters to avoid repetitive "presents" pattern.

### Chapter Introduction Alternatives

| Chapter | Verb Options |
|---------|--------------|
| Introduction | introduces, outlines, establishes, sets out |
| Literature Review | reviews, examines, surveys, analyses, discusses |
| Methodology | describes, outlines, explains, details, presents |
| Results | reports, presents, shows, details, summarises |
| Discussion | discusses, analyses, interprets, examines, considers |
| Conclusion | concludes, summarises, synthesises |

### Sentence Structure Variations

| Repetitive Pattern | Varied Alternatives |
|-------------------|---------------------|
| Chapter 2 presents the literature review | Chapter 2 reviews relevant literature |
| | The literature review in Chapter 2 examines... |
| | Relevant literature is reviewed in Chapter 2 |
| Chapter 3 presents the methodology | The methodology is described in Chapter 3 |
| | Chapter 3 outlines the research methodology |
| | Methodological details appear in Chapter 3 |
| Chapter 4 presents the results | Results are reported in Chapter 4 |
| | Chapter 4 details experimental results |
| | Experimental findings are presented in Chapter 4 |
| Chapter 5 presents the discussion | Chapter 5 discusses these findings |
| | The discussion in Chapter 5 analyses... |
| | These results are discussed in Chapter 5 |

### Research Question Verb Variations

| Standard Pattern | Alternatives |
|-----------------|--------------|
| RQ1 addresses | RQ1 investigates, examines, explores |
| RQ2 investigates | RQ2 asks, considers, analyses |
| RQ3 explores | RQ3 determines, assesses, evaluates |

### Research Objective Verb Variations

| Standard Pattern | Alternatives |
|-----------------|--------------|
| RO1 aims to develop | The first objective is to develop |
| RO2 aims to evaluate | Evaluation of X forms the second objective |
| RO3 aims to propose | The third objective proposes... |

### Research Contribution Variations

| Standard Pattern | Alternatives |
|-----------------|--------------|
| RC1: This research contributes | The first contribution is... |
| RC2: This research contributes | A second contribution involves... |
| RC3: This research contributes | This research also contributes by... |
